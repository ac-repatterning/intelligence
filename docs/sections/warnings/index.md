# warnings

This pages publishes river level forecasts per gauge station that lies within a <a href="https://weather.metoffice.gov.uk/warnings-and-advice/uk-warnings" target="_blank">weather warning area</a>.  The National Weather Service provides United Kingdom weather warnings, therefore we have set up a program that automatically scans for National Weather Service warnings.

If one or more SEPA gauge stations, on record, lie within a warning area, a temporary schedule is automatically created. Every $N$ hours the schedule automatically instigates river level forecasting per affected gauge station.  This continues until the end of the warning period.

For more about an inference model's attributes, study the [expert section](../expert/forecasting.md){ data-preview }.
